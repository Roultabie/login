#Login

*A simple script to create resctricted areas in your projects*

##How to use :

Download login and login form in your script dir, create a global config user and password (like $config['user'] $config['hash'] $config['salt'] $config['sessionExpire'] - (use static method returnHash() to generate your hash) - and include login.php in your page(s).  
That's all !
